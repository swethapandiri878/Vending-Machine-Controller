# Vending Machine Controller – Verilog HDL

## Overview
Designed and simulated a Moore FSM-based vending machine controller using Verilog HDL to handle product selection, coin detection, and change return logic.

## Features
- Accepts ₹5 and ₹10 coins
- Product cost: ₹15
- Automatic change return
- Moore FSM implementation
- Fully synthesizable RTL

## State Diagram
Idle → ₹5 → ₹10 → Vend

## Tools Used
- Verilog HDL
- Vivado Simulator
